Project Explanation

This project improves the accuracy of prediction for Premier League football matches. The project uses a machine learning algorithm, which learns from data sets. These data sets include statistical information and results. The project is completely cloud based. TellMeTheResult.com is a purchased URL from GoDaddy.com where the user can access the information. This URL is being hosted from a webserver called Tomcat. It uses technologies such as Python, Java, Google Cloud, SQL and much more.
