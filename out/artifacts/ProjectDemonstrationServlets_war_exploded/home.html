Final Year Project GMIT

The title of my final year project is Football Prediction Analysis with Machine Learning. This project has been developed using the skills I've acquired over the last five years of doing Computer and Electronic Engineering.

This project looks to improve the accuracy of prediction analysis for Premier League football machines using Machine Learning. The machine will learn from data sets which include historical results and statistical information relating to the Premier League. This project is being developing during the 2017/18 season. It is completely cloud based. The webpage TellMeTheResult.com is a domain name purchased from GoDaddy.com and hosted by a webserver technology called Tomcat.
